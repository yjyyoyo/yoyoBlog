# 一、Shiki Highlighter 代码块美化

- 底层：VS Code 同款 Shiki 着色引擎，色彩分层极细腻，关键字 / 类型 / 注释 / 字符串区分度拉满community....
- 全模式统一：**源码模式、实时预览、阅读模式三色完全一致**，不再出现编辑和预览两种颜色
- 主题库：内置 Catppuccin、Nord、Tokyo Night、One Dark Pro、Gruvbox、GitHub 黑白等 60+ VS Code 官方配色；支持导入本地 VS Code 主题 json
- 高级功能：
    
    - 行高亮（单行 / 区间标记重点代码）
    - 代码块顶部标题、语言图标
    - 行号、代码折叠、diff 增减行标记
    - 行内代码`{js} xxx`也能着色
    - 支持 100 + 编程语言（含小众脚本、配置文件、SQL、正则）
- 缺点：插件体积略大；老设备实时预览轻微掉帧
- 适合：写技术笔记、编程文档、追求 VS Code 同款视觉、需要编辑 / 预览统一着色

# 二、Excalidraw 插件 (画板实时Web 渲染)
- 如果你重度依赖画板，并且非常希望在网页上展示，建议在 Obsidian 中放弃自带的 Canvas，改用 Excalidraw 插件。
- Excalidraw 也是 Obsidian 里非常强大的白板工具。
- 它的核心优势是：Excalidraw 插件可以设置为自动将画板导出为 SVG 或 PNG 图片，并且这张图会和原本的画板文件实时同步更新。
- 只要你在 Markdown 里链接这个 SVG，当你 push 到 Quartz 时，它就能完美、清晰地显示在网页上。
