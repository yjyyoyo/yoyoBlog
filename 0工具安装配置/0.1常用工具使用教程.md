## 将obsidian笔记同步到Github上
- 在Github上建立空仓库
- 将本地仓库变为Git仓库
```C
PS C:\Users\13956\Desktop\中医\中医笔记> git init
Initialized empty Git repository in C:/Users/13956/Desktop/中医/中医笔记/.git/
PS C:\Users\13956\Desktop\中医\中医笔记>
```
- 打包本地文件(add,commit,branch)
	- add将文件添加到暂存区  `git add .`
	- commit提交文件 `git commit -m "首次提交我的Obsidian笔记`
	- branch将默认的主分支名命名为main `git branch -M main`
- 连接并推送到Github(remote,push)
		- 绑定本地仓库和Github云仓库:`git remote add origin https://github.com/yjyyoyo/Traditional-Chinese-medicine-notes.git`
		- 将本地文件推送到云端:`git push -u origin main`
